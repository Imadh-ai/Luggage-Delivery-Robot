An intelligent embedded robotic system designed to autonomously carry passenger luggage and guide users to predefined airport destinations. The system uses line-following navigation, RFID-based destination selection, and real-time load monitoring to provide a smart, low-cost airport automation solution.

---

## 📌 Overview

This project presents a **smart airport assistance robot** that reduces passenger effort by transporting luggage and navigating inside airport environments autonomously.

The robot follows predefined paths using IR sensor-based line tracking, identifies destinations using RFID authentication, and measures luggage weight using a load cell system.

It demonstrates practical implementation of **embedded systems, sensor fusion, and autonomous navigation** in real-world applications.

---

## 🚀 Key Features

- ✅ Autonomous line-following navigation system  
- ✅ RFID-based destination selection  
- ✅ Real-time luggage weight measurement  
- ✅ Load Cell + HX711 weight sensing module  
- ✅ Predefined route guidance system  
- ✅ Battery-powered portable robotic platform  
- ✅ Modular embedded system design  
- ✅ Low-cost airport automation prototype  

---

## ⚙️ System Workflow

1. System initializes all sensors and modules  
2. Passenger places luggage on the robot platform  
3. Load cell measures the weight of luggage  
4. RFID card is scanned to select destination  
5. Robot processes navigation route  
6. IR sensor array detects and follows the line path  
7. Robot reaches the selected destination autonomously

## 🛠️ Hardware Components

- Arduino UNO  
- NodeMCU ESP8266  
- L298N Motor Driver  
- IR Sensor Array (Line Following)  
- RFID RC522 Module  
- Load Cell (1kg) + HX711 Amplifier  
- DC Geared Motors  
- Li-ion Battery Pack with BMS  
- DC-DC Buck Converter

## 📡 System Architecture

- **Controller Unit:** Arduino UNO (main logic & motor control)  
- **Communication Unit:** NodeMCU ESP8266 (optional IoT expansion)  
- **Sensing Unit:** IR sensors + Load cell + RFID module  
- **Actuation Unit:** L298N motor driver + DC motors
